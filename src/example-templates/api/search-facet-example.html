import { CutSearchModule } from 'cut';

Main/Root level Component

Directive

Selector: cut-search-facets

This will give you full component, contains all facets, listed below. all are driven/controlled by the data provided as an input.

<cut-search-facets> is used for whole facet module 

<cut-search-facets [facetData]="facetData" (facetEventHandler)="getChangedEvent($event)"></cut-search-facets>

Properties

Name Description
@Input() 

facetData: (
  InputDropDownModal |
  InputModal |
  CheckListDataModal
  )[]

Collection of multiple object. each object is used to define the facet component with respect to their data model
@Output() 

facetEventHandler: EventEmitter<
  InputDropDownFacetModal |
  SingleValueModel |
  ArrayValueModel
  )[]>

Collection of multiple object. each object is defined by the specific model of different facet component output.

Events

The getChangedEvent output event is emitted when the values changes.

Data Models-(Interface)

InputDropDownModal - same will work as an input for individual component.

Key/Type Description
showFacet: boolean Whether the component is displayed or not.
autoCompeleteList: string[] List to populate the auto suggestion.
facetTitle: string Facet title.
inputTitle: string Title for the input field.
dropDownOptions: DropDownModel[] Privide the adta for dropdown.
defaultMustHave: number Values : Enum - 1,2,3, used as the default value.
type: string Values : 'inputDropDownValues' must be provided.
fieldToWorkWith: string To identify the output emited by the component..

CheckListDataModal - same will work as an input for individual component.

Key/Type Description
showFacet: boolean Whether the component is displayed or not.
facetTitle: string Facet title.
optionsList: CheckListModel[]; Provide the data for list items.
defaultMustHave: number Values : Enum - 1,2,3, used as the default value.
type: string Values : 'inputDropDownValues' must be provided.
fieldToWorkWith: string To identify the output emited by the component..

InputModal - same will work as an input for individual component.

Key/Type Description
showFacet: boolean Whether the component is displayed or not.
autoCompeleteList: string[] List to populate the auto suggestion.
facetTitle: string Facet title.
defaultMustHave: number Values : Enum - 1,2,3, used as the default value.
type: string Values : 'inputDropDownValues' must be provided.
fieldToWorkWith: string To identify the output emited by the component..

DropDownModel

Key/Type Description
displayName: string Values to be displayed.
displayValue: string value to be used.
isSelected: boolean If the value is been selected on not.
