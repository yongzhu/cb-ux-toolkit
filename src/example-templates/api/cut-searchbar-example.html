API reference for Cut Search Bar Component

import { CutSearchModule } from 'cut';

Main/Root level Component

<cut-searchbar (resultDispatch)="searchResult($event)" [searchapi]="api_url" [apiData]="apiData"></cut-searchbar> creates an input and search button.

Attributes

(resultDispatch)" is the output from the api search result. Returns the remapped result from the API. searchResult would be a function you create to react to the API response from each search.

[searchapi] is the url to call with the search string which will respond with the matching records.

[apiData] is an object with keys: params and config - param contains all the parameters for query string(they will be appended to the url.) and config contains your custon header configuration.
