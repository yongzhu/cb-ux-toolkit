import { CutSearchModule } from 'cut';

Main/Root level Component

Directive

Selector: cut-search-result

This will give you full component, contains all facets, listed below. all are driven/controlled by the data provided as an input.

<cut-search-result> is used for whole facet module 

<cut-search-result [resultData]="resultData" (detectChanges)="detectChanges($event)"></cut-search-result>

Properties

Name Description
@Input() 

facetData: CutResultModel

Collection of multiple object. each object is used to defines a component with respect to their data model
@Output() 

detectChanges: EventEmitter<{
changeType: "'tab-change'/'type-change'",
id: id
}>

this may be changed as in future update, please checkout the output before use.

Events

The detectChanges output event is emitted when the values changes.
