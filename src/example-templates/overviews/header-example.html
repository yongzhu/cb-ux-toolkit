<cut-header> is a circular indicator to show activity.

View mode

The loading indicator can be shown by setting the "loading" property to true, note this property is disabled by default. <cut-header>.

Theming

The color of a loading-indicator can be changed by using the color property. By default, the loading-indicator uses the theme's primary color. This can be changed to 'accent' or 'warn'.

Accessibility

Each loading-indicator should be given a meaningful label via aria-label or aria-labelledby.
