<cut-searchbar> search bar tool that can be used throughout CB product sites.

View mode

The searchbar takes an API URL [searchapi] to call with the search and a map [map] which is a key to indiate which mapping to use to ingest the API result. <cut-searchbar [searchapi]="exampleApis.test_nolocal" [apiData]="apiData">.
