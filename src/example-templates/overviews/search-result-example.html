API reference for Cut Search Component

import {CutSearchResultModule} from 'cut';

Main/Root level Component

<cut-search-result> is used for whole result module

Individual Components

<cut-used-filter> is used to get the used filter for search result module.

<cut-result-tabs> is used to get the used filter for search result module.

<cut-result-type-filter> is used to get the used filter for search result module.

<cut-result-filters> is used to filter the search result.

<cut-result-list> is used to rander the result list.
